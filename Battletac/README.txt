------------------- Battletac -------------------
| Developed and Created By Brady Schnell        |
| Contact: bradyschnell@gmail.com
-------------------------------------------------


Tic-tac-toe based stratigie multiplayer game where the objective of the game
is to get a certain number of markers in a row. Both players are able to use a variety of
items that can help them win the game, and prevent the other player from winning.

The game is designed for both Player vs Player and Player vs Comp. 

Development status: Work in progress, AI currently in development.
